# Gestor de Funcionários em Python (Versão Básica)

Aplicação em Python para gerenciamento simples de funcionários, utilizando listas paralelas para armazenar dados.

## Funcionalidades

- Adicionar funcionário
- Excluir funcionário por índice
- Listar funcionários cadastrados
- Encerrar o programa
- Validação para lista vazia

## Estrutura dos dados

Os dados são armazenados em listas separadas:
- `nomes`
- `cargos`
- `salarios`

Cada índice representa um funcionário.

## Menu

1 - Excluir funcionário  
2 - Adicionar funcionário  
3 - Listar funcionários  
4 - Sair  

## Objetivo

Exercício introdutório para prática de:
- Listas
- Condicionais
- Laços de repetição
- Entrada e saída de dados

## Observações

- Não há persistência de dados.
- Os dados são perdidos ao encerrar o programa.
- Projeto educacional.
